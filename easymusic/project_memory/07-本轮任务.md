# 本轮任务

## 使用方式

这个文件只记录“当前正在推进的单个任务”。

和其他文件的区别：

- `01-当前状态.md` 记录项目整体状态
- `06-待办清单.md` 记录中短期待办列表
- `07-本轮任务.md` 只记录这一轮正在做的最具体任务

每次开始新任务时，应更新本文件；每次结束当前会话前，如果任务未完全结束，也要更新本文件，保证下次能直接接上。
