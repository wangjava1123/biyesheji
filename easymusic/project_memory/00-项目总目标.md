# 项目总目标

## 项目背景

当前项目基于现有源码进行二次开发，用于毕业设计。目标不是简单复用原项目，而是在原有 AIGC 音乐平台基础上形成真实的个人改造和实现成果。

## 当前统一目标

项目主线统一为：

**用户一句话描述需求 -> AI 扩写成结构化提示词 -> 生成音乐 -> AI 生成封面 -> 积分结算 -> 作品发布与统计**

## 本次改造的核心方向

1. 增加自然语言驱动的智能提示词增强能力
2. 将增强后的提示词接入音乐生成主链路
3. 新增 AI 封面生成能力
4. 打通提示词、音乐、封面的积分扣减与失败退款
5. 增加作品发布、隐藏、统计等生命周期管理
6. 优化当前项目中的低成本不合理点和界面问题

## 成果定位

最终希望把项目升级为：

**基于 Spring Boot + Vue 的 AIGC 音乐生成与多模态内容发布平台**
